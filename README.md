# CHZ Burn Tracker

Live tracker for CHZ value accrual under Tokenomics 2.0.

- Live price chart powered by [CoinGecko](https://www.coingecko.com/en/coins/chiliz) (7D / 30D / 180D / MAX).
- Cumulative and monthly burn breakdowns.
- Burn source attribution: Fan Token revenue buyback, EIP-1559 gas burns, Fan Token Play.
- Inflation decay schedule and projected net supply path.
- Annotated price-vs-burn impact chart for the [Tokenomics 2.0 announcement](https://x.com/Chiliz/status/2027103512409051232).
- Per-chart toggle: CHZ amount or USD value.

Static HTML, no build step. Deployed via GitHub Pages.
